# Notebooks

This directory contains the Jupyter notebooks for the EuBIC 2026 Winter School.

## Notebooks

- `EUBIC_Task1_Peaks.ipynb` - Task 1: Peaks analysis
- `EUBIC_Task2_ID.ipynb` - Task 2: Identification
- `EUBIC_Task3_Quant.ipynb` - Task 3: Quantification

## Running Notebooks

From the root of the repository, start Jupyter:

```bash
jupyter lab
```

Then navigate to this directory and open any notebook.
